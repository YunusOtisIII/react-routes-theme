## Quick Start (Local)

```bash
# 1) Clone
git clone https://github.com/a2rp/react-routes-theme.git
cd react-routes-theme

# 2) Install
npm i

# 3) Dev
npm run dev
# open: http://localhost:5173/react-routes-theme

# 4) Build (static)
npm run build

# 5) Preview production build
npm run preview
# open: http://localhost:4173/react-routes-theme
```
